- **Type / Category** — taxonomy
- **Marketplace Evidence** — gerçekten satılıp alınmış benzer asset'lerin pattern'i

- **Typical MRR Band** — gelir aralığı
- **Time to First Dollar** — ilk gelir süresi
- **Mission / Vision** — niyet çerçevesi
- **Strategic Themes** — Allocator'ın OKR turetmesi için weighted axes (0-100)
- **Recommended Blueprints** — sales-marketing / customer-success / product

## Detaylı Template'ler

### 🔹 SaaS Ürünü (`saas`, software)

**Tagline:** B2B abonelik yazılımı — MRR + CAC + churn üçgeni.

**Marketplace Evidence:** Acquire.com + TrueMRR top segment — $10-50K MRR bandında sürekli deal akışı.

**Typical Multiple:** 3-5x MRR
**Typical MRR Band:** $5K-$50K/ay
**Time to First Dollar:** 4-9 ay

**Mission:** [Sektör] profesyonellerinin günlük iş akışını 10x hızlandırmak.
**Vision:** 5 yıl içinde sektörün de-facto işletim sistemi haline gelmek; NRR 115%+ ile kapalı, sürdürülebilir büyüme.

**Strategic Themes (weight):**
- Müşteri Başarısı (95) — NRR > 115%
- Ürün Kaldıracı (80) — Her feature workflow'a dönüşür
- Dikey Uzmanlık (70) — Sektör dilini ana dilim gibi konuşurum

**Recommended Blueprints:** sales-marketing, customer-success, product

## Oracle Pick Rules — Portfolio Gap Recommendations

Allocator yeni asset önerisi yaparken Matrix'in `oraclePicksForPortfolio()` rule engine'i:

```
Mevcut portföye bak:
  hasSoftware = saas|software|app|extension içeriyorsa
  hasContent  = content|newsletter|podcast|youtube|course|media
  hasCommerce = commerce|d2c|pod|affiliate|digital product|templates
  hasService  = service|agency|community|job

Önceliklendirilmiş öneriler:
  1. !hasContent          → Newsletter (HIGH) — compounding dağıtım kanalı
  2. !hasSoftware         → Micro-SaaS (HIGH) — recurring revenue omurgası
  3. !hasCommerce + hasContent → Affiliate (MEDIUM) — niş ortak, SEO overlap
  4. hasContent + !youtube → YouTube (MEDIUM) — attention moat compounding
  5. hasSoftware + hasContent + !hasService → Community (LOW) — peer-driven CS
  6. fallback (boş)        → Podcast (MEDIUM) — yavaş ama derin bağlanır
```

**Output: Top 3 picks, priority+reason ile.**
